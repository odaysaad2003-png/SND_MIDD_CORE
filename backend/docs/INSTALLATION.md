1. Copy all approved `.md` files from this folder into `backend/docs/`, except:
   - `AI_PROJECT_INSTRUCTIONS.md`
   - `SPRINT_PROMPT_TEMPLATE.md`
   - `DOCUMENTATION_REVIEW.md`
2. Put `AI_PROJECT_INSTRUCTIONS.md` and `SPRINT_PROMPT_TEMPLATE.md` in the backend root.
3. Keep `DOCUMENTATION_REVIEW.md` outside the active source set or archive it after review.
4. Delete/archive the old `PROJECT_STATE_UPDATED.md`.
5. Replace the old accumulated `PROJECT_STATE.md`.
6. Rename/remove `CLAUDE_PROJECT_INSTRUCTIONS.md` to avoid duplicate instruction sources.
7. Add the approved active files to ChatGPT/Claude project sources.
8. After adding the full backend code, perform one final consistency check for exact Report
   route names and status/reason/action enums.
